INSTRUCTIONS

  • This app helps yout to keep record of your daily routine's time consumption
  • Enter a NAME of your routine task and click SELECT to register
  • Alternatively, you can type-in and SELECT a routine name from the already registered DATA
  • To punch-in / punch-out a routine timer, click the START / STOP button which would appear after the SELECT click
  • You can have any number of routine timer to run at the same time
  • Use the RESET button to shift between routines
  • You can analyse the recorded routines from the DATA tab
  • You need to be connected to the internet only during the START / STOP / DELETE operations. Meanwhile, the records will be preserved in the database
